Principais Recursos

O veículo robô Arduino é equipado com uma variedade de recursos para proporcionar uma experiência completa. Além dos movimentos básicos de andar para frente, para trás e virar para os lados, o robô apresenta um sensor frontal de distância para detectar obstáculos, uma luz e sinal sonoro de ré para operações reversas e um LED RGB que indica seu estado de conexão à API - verde para conectado e vermelho para desconectado.

Apresentação do Projeto

Assista à nossa apresentação em vídeo para ter uma visão detalhada do projeto. Neste vídeo, explicamos a estrutura do robô Arduino, seus recursos, a API HTTP desenvolvida para comunicação e a interface de programação em blocos visuais usando a biblioteca Blockly. É uma oportunidade para conhecer de perto o funcionamento e a inovação por trás do projeto.

Documentação da API

Explore nossa documentação da API para entender como interagir e enviar comandos para o veículo robô Arduino. A documentação fornece detalhes sobre os endpoints, parâmetros e respostas, tornando mais fácil para desenvolvedores integrarem novas funcionalidades ou aprimorarem o sistema existente.

Interface de Programação Blockly

Nossa interface de programação em blocos visuais, baseada na biblioteca Blockly, oferece uma maneira intuitiva de controlar o veículo robô. Com blocos personalizados, você pode criar lógicas complexas sem a necessidade de programação tradicional. Experimente a interface para vivenciar a simplicidade e eficiência da programação visual.

Desafios do Mundo Real

Nosso projeto não se limita apenas à teoria. Para simular desafios do mundo real, o professor criará pistas no laboratório com obstáculos. Sua tarefa será desenvolver algoritmos de blocos que permitam ao robô percorrer autonomamente essas pistas, demonstrando a eficácia e aplicabilidade prática do sistema.
